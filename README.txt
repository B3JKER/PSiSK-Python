PSiSK
Klasyfikacja danych medycznych siecią głebokiego uczenia - Python, KERAS

W celu użycia programu należy:
	1. Pobrać wszystkie pliki z repozytorium PSiSK-Python.
	2. Utworzyć nowy projekt w wybranym kompilatorze języka Python (w naszym przypadku był to PyCharm)
	3. Dodać do projektu pobraną zawartość.
	4. Pobrać wszystkie niezbędne biblioteki i moduły (patrz plik requirements.txt)
	5. Uruchomić plik 'main.py' w celu przeprowadzenia procesu uczenia. Poczekać aż program zakończy działanie.
	6. Uruchomić symulację wybranego pacjenta w aplikacji pobranej z repozytorium PSiSK-front.
	7. Uruchomić plik 'read.py' w celu rozpoczęcia połączenia programu z symulatorem i generowania diagnoz.
	8. Diagnoza jest przesyłana do symulatora co 10 sekund, a dane pobrane z symulatora w postaci utworzonego
	   wykresu są zapisywane w folderze projektu w pliku 'data.png'.
